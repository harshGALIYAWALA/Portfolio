Safe Track

Kotlin Google Maps API Node.js Express.js MongoDB Firebase WebSocket real-time GPS

Safe Track is designed to provide enhanced security and peace of mind for women while traveling. It features real-time location tracking, route monitoring, and emergency alerts to trusted contacts. The app includes a "geoguessing" feature that tracks the user's route and triggers alerts if they deviate or remain stationary for too long. Additional safety features include a "Time Threshold Alert" for delays, smart route deviation detection, and an emergency response system if the user is unresponsive. It also offers a "Fake Shutdown Mode" to secretly send emergency data in case of an emergency.
